### Дилемма 

Если мы  
разделяемы плоскостью,  
если   
друг есть,  
есть враг,  
есть правый,  
  
Если всё  
решается косинусом,  
если   
инструкция есть  
к деталям,  
  
Если мир  
черно-бело полюсный,  
если   
плюс есть,   
есть минус,  
есть грани,  
  
Тогда  
отвечайте полностью,  
следя,  
чтобы   
себе не соврали:  
  
Как   
договориться с совестью,  
не вызвав подозрения  
у морали?  
  
21.02.2024
